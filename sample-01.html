sample-01：バインディングとイベント

headにCDNで開発バージョンのVue2を読み込む。※公開時にはエラー出力されない本番用を使う

#appの中がvueの対象となる

{{テキスト}}でバインディング（※テキストのみ、属性値には使えない）

v-model="リアクティブ・データ"で双方向バインディング

v-on:click="メソッド名"でクリックイベント

vueのスクリプトは、HTMLの最後に

el: Vueアプリ登録

data: リアクティブ・データ登録（ここに登録されたデータがバインディング可能になる）

methods: メソッド（処理）登録


var app = new Vue({
	el: '#app',
	data:{
		fruit1: 'りんご',
		fruit2: 'バナナ',
		fruit3: 'みかん',
		vegetable:{
			item1: '白菜',
			item2: '人参',
			item3: '水菜',
		}
	},
	methods:{
		click_btn: function(){
			this.vegetable.item1 = 'じゃがいも';
			this.vegetable.item2 = 'ホウレンソウ';
			this.vegetable.item3 = 'もやし';
		}
	}
});
